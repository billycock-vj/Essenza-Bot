# Instrucciones para Imagen Base

## Ubicación de la Plantilla

La imagen base (plantilla) debe colocarse en:

**Windows (desarrollo local):**
```
C:\apps\essenza-bot\data\imagenes\plantilla_base.png
```

**Fly.io (producción):**
```
/data/imagenes/plantilla_base.png
```

## Requisitos de la Imagen

- Formato: PNG
- Tamaño recomendado: 800x1200 píxeles (o proporción similar)
- La imagen se usará como fondo para todas las citas generadas
- El texto se superpondrá sobre esta imagen

## Crear el Directorio

El directorio se crea automáticamente al ejecutar el bot, pero puedes crearlo manualmente si es necesario.
